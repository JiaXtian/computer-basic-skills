# Shell 速查

| 场景 | 命令 |
|---|---|
| 查看目录 | `ls -alh` |
| 查找文件 | `find . -name "*.log"` |
| 文本筛选 | `grep -R "ERROR" .` |
| 查看进程 | `ps aux` |
| 杀进程 | `kill -15 <pid>` |
| 权限修改 | `chmod 755 script.sh` |
