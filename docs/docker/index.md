# Docker 模块总览

Docker 让应用运行环境可复制、可迁移、可隔离，是现代工程交付的标准能力。

## 学习目标

- 理解镜像/容器/网络/卷关系
- 能编写 Dockerfile 并优化镜像体积
- 能使用 Compose 管理多服务

## 章节安排

1. L1：镜像与容器基础
2. L2：Compose 编排与环境管理
3. L3：多阶段构建 + 实战项目
4. 排错：端口冲突、卷挂载、构建缓存问题
