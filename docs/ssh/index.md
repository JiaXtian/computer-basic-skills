# SSH 模块总览

SSH 是远程管理服务器和安全访问 Git 平台的核心能力。

## 学习目标

- 理解密钥认证机制与主机信任模型
- 掌握多账号、多主机配置
- 熟练使用端口转发进行调试与安全访问

## 章节安排

1. L1：密钥、agent、known_hosts、config
2. L2：安全加固、跳板机、端口转发
3. L3：GitHub 多账号与远程调试实战
4. 排错：握手失败、权限错误、主机指纹冲突
