# 学习路径

## 阶段划分

| 阶段 | 目标 | 章节 | 难度标签 |
|---|---|---|---|
| 阶段 1 | 建立命令行与版本控制基础 | Shell L1/L2 + Git L1 | L1 入门 |
| 阶段 2 | 建立协作与远程连接能力 | Git L2/L3 + SSH L1/L2 | L2 进阶 |
| 阶段 3 | 构建容器化工程能力 | Docker L1/L2/L3 | L2/L3 |
| 阶段 4 | 完成工程闭环 | Linux/Network/CI + 综合项目 | L3 实战 |

## 每周建议节奏

- 周一到周三：学习章节与命令练习
- 周四：完成实战任务
- 周五：做排错题与自测题
- 周末：复盘并整理速查笔记

## 验收标准

- 能独立完成一次 Git 分支协作与 PR 提交流程
- 能使用 SSH 安全连接远程主机并进行基本诊断
- 能编写 Dockerfile 并用 Compose 启动多服务
- 能读懂基础日志并定位常见错误
