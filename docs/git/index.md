# Git 模块总览

Git 是现代软件协作的基础。你将从“本地版本控制”过渡到“团队协作与历史治理”。

## 学习目标

- 理解提交历史与快照模型
- 熟练使用分支与合并策略
- 能修复常见误操作并恢复历史

## 章节安排

1. L1：仓库、提交、日志、差异
2. L2：分支协作、rebase、PR 工作流
3. L3：从零完成一次团队协作演练
4. 排错：误删提交、冲突、历史污染修复
