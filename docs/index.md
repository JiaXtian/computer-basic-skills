# 计算机基础技能文档站

这是一套面向学习者的系统化教程，目标是让你通过连续阅读与实操，完整掌握开发运维基础能力，并建立机器学习与大语言模型方向的系统认知。

整站按照“技术主题”组织为长章节，左侧导航可直接定位到对应主题，章节内部提供目录与命令示例，便于边读边练。

## 章节总览

| 章节 | 你将掌握什么 |
|---|---|
| [Git 完整指南](git.md) | 版本控制模型、分支协作、冲突处理、历史恢复、发布管理 |
| [SSH 完整指南](ssh.md) | 密钥认证、远程连接、跳板机、端口转发、安全加固与排障 |
| [Docker 完整指南](docker.md) | 镜像与容器、Dockerfile、Compose、资源限制、Kubernetes 入门 |
| [Linux 与 Shell 指南](linux.md) | Linux 系统基础、权限与进程、日志排障、Shell 脚本实践 |
| [网络故障诊断指南](network.md) | DNS/TCP/HTTP/TLS 基础、抓包方法、分层排障与性能定位 |
| [Python 完整指南](python.md) | 环境管理、语言基础、依赖治理、代码规范与工程实践 |
| [机器学习完整指南](ML.md) | 机器学习核心范式、经典算法、模型评估与调优、工程实践 |
| [强化学习完整指南](RL.md) | MDP、价值函数、策略优化、DQN/PPO/RLHF 等核心方法 |
| [深度学习完整指南](DL.md) | 神经网络原理、训练机制、CNN/RNN/Transformer 与工程技巧 |
| [大语言模型完整指南](LLM.md) | 预训练、对齐微调、RAG、推理优化与 Agent 应用路径 |
| [GitHub 使用指南](github-guide.md) | 仓库协作、Issue/PR、权限管理、Actions 与 Pages 发布 |

## 推荐阅读顺序

1. Git -> GitHub（先掌握协作基础）
2. Linux 与 Shell -> SSH（先掌握主机与远程操作）
3. Docker -> 网络（进入部署与排障体系）
4. Python（用于自动化与工程开发补强）
5. 机器学习 -> 深度学习 -> 大语言模型（建立 AI 主线能力）
6. 强化学习（作为进阶方向，补齐决策学习范式）

## 使用建议

- 每章建议先按目录完整通读一遍，再做命令实操。
- 遇到报错时，优先回看对应章节的“排错/诊断”部分。
- 建议把高频命令做成个人速查笔记，形成自己的学习闭环。
