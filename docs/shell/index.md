# Shell 模块总览

Shell 是你与操作系统交互的最直接接口。掌握 Shell 后，你可以把重复性操作自动化，并快速排查问题。

## 学习目标

- 理解 Bash 核心语法与命令组合方式
- 了解 `bash/zsh/fish/powershell` 的关键差异
- 能独立编写可复用脚本

## 章节安排

1. L1：变量、路径、重定向、管道、权限
2. L2：函数、条件、循环、进程、任务控制
3. L3：自动备份、日志分析、批量处理实战
4. 排错：环境变量、权限、换行符、路径问题

## 模块验收

完成本模块后，你应能：

- 在 15 分钟内写出一个可执行脚本并通过参数接收输入
- 使用管道与 `grep/awk/sed` 对日志做筛选
- 在错误发生时用 `set -x` 与退出码定位问题
