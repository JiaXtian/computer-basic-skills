# 计算机基础技能文档站

本项目是一套持续补充的系统化技术教程，覆盖开发运维基础与 AI 基础学习路线。内容以“完整章节 + 原理讲解 + 命令示例 + 排错思路”为核心组织方式，适合日常学习与查阅。

在线阅读（GitHub Pages）：
[https://jiaxtian.github.io/computer-basic-skills/](https://jiaxtian.github.io/computer-basic-skills/)
