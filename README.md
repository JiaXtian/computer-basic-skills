## 适合谁阅读

- 计算机专业学生
- 转行工程师
- 需要补齐工程基础的开发者
- 希望系统掌握命令行、协作与部署基础的学习者

## 学习建议

1. 按首页推荐顺序阅读，先打通协作与主机操作主线。
2. 每章阅读后立即做命令实操，优先形成肌肉记忆。
3. 遇到错误时先看章节中的排障方法，再做外部检索。
